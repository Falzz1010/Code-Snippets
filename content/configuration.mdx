# Configuration

Learn how to configure Code Snippets for your needs.

## Environment Variables

Create a `.env.local` file with these variables:
```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-key
```

## Theme Configuration

You can customize the theme in `theme.config.js`:

```javascript
export default {
  darkMode: true,
  syntaxHighlighting: {
    theme: 'github-dark',
    languages: ['javascript', 'python', 'rust']
  }
}
```
