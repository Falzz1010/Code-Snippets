# Basic Usage

Learn how to use Code Snippets effectively.

## Creating a Snippet

1. Click "New Snippet" button
2. Enter your code in the editor
3. Select the programming language
4. Add a title (optional)
5. Click "Create"

## Sharing Snippets

1. Open any snippet
2. Click the "Share" button
3. Choose sharing method:
   - Copy link
   - Share on Twitter
   - Share on LinkedIn

## Managing Snippets

- View all snippets on your dashboard
- Edit snippets using the edit button
- Delete unwanted snippets
- Track version history
